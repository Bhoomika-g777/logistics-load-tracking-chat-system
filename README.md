# Logistics Load Tracking Chat System

## Overview

This project is a **chat-based logistics load tracking system** built using **Node.js, Express, HTML, CSS, and JavaScript**.
The system allows users (such as brokers or customers) to interact with a **menu-based chatbot interface** to retrieve shipment information and test API integrations.

The project demonstrates how a **chat interface can simplify load tracking and logistics queries**.

---

## Features

* PIN-based authentication for accessing the chat portal
* Menu-based chatbot interaction
* Load tracking interface
* API integration for retrieving data
* Command-based chatbot responses
* Modular project structure
* Node.js backend servers

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/logistics-load-tracking-chat-system.git
```

2. Install dependencies

```
npm install
```

3. Run the server

```
node server.js
```

4. Open the HTML file in your browser to access the chat interface.
